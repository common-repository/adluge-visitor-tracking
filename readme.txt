This plugin allows you to add the AdLuge tracking script to every page of your website.

== Description ==

The AdLuge Visitor Tracker  is a simple wordpress plugin that lets you add the AdLuge visitor tracking script to WordPress sites without having to get your hands dirty in code. Moreover, the simple interface gives you one place where you can edit the necessary pages needed to be tracked.The AdLuge tracking script will be added to the footer during plugin installation and will be enabled automatically. 

Note for Beginners: Like all plugins available for download, AdLuge Visitor Tracker is only available for self-hosted WordPress sites. 

For more information visit [AdLuge](http://www.adluge.com)
== Installation ==

1. Upload `AdLuge Visitor Tracker` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin (go to the Settings/AdLuge Visitor Tracker)
4. Make sure wp_footer() is called in common file 
5. Enjoy :)
